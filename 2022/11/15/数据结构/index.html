常用的数据结构：数据表、链表、栈、队列、树、哈希表

顺序表

优点 ：因为顺序表是通过数组实现的，所以在内存上是连续，通过下标就可以直接访问数据的内部成员
缺点 ：在插入和删除的时候，需要移动大量的数据，所以效率比较低，而且长度是固定的，不方便后续的扩展

链表

单向链表

优点：单向链表增加删除节点的效率比较高，因为只需要改变指针的指向就可以了，不需要移动大量的数据，并且遍历的时候不会死循环

缺点：只能从头到尾的遍历，只能找到后继，不能找到前驱

树

完全二叉树

性质：

  1. 二叉树第 i（i≥1） 层上的节点最多为 2^(i-1) 个。
  2. 深度为 k（k≥1） 的二叉树最多有 2k-1 个节点。
  3. 在任意一棵二叉树中，树叶的数目比度数为 2 的节点的数目多一。
    • 总节点数为各类节点之和：n = n0 + n1 + n2
    • 总节点数为所有子节点数加一：n = n1 + 2*n2 + 1
    • 故得：n0 = n2 + 1 ;
  4. 满二叉树 ：深度为 k（k≥1） 时有 2^k－1 个节点的二叉树。
  5. 完全二叉树 ：只有最下面两层有度数小于 2 的节点，且最下面一层的叶节点集中在最左边的若干位置上。具有n个节点的完全二叉树的深度为
    • （log2n）＋1或log2(n+1)。

完全二叉树的遍历

  1. 先让根节点入队，让根节点出队，打印根节点的值，
  2. 判断根节点是否有左右孩子，如果有左右孩子让他们入队。
  3. 先让根节点的左孩子出队，打印左孩子的值
  4. 在判断它是否有左右孩子，如果没有就不入队了。
  5. 先让根节点的右孩子出队，打印右孩子的值，
  6. 判断它是否有左右孩子，如果没有就不入队了。
  7. 此时队列为空，循环退出即可。

二叉树节点插入

  1. 首先将待插入结点的值与根结点的值作比较，若val == root->data，此时我们直接返回当前root指针，因为我们规定，二叉排序树中不存在值相同的结点。
  2. 若待插入结点的值小于根结点的值，此时我们应该进入根结点的左子树
  3. 若待插入结点的值大于根结点的值，此时我们应该进入根结点的右子树
  4. 若指针为空，此时我们根据传入的值创建结点，并返回创建的结点指针

#define N 13 //  11/0.75 = 14.6取最大质数
#define datatype int
typedef struct node {
    datatype data;
    struct node* next;
} hash_t;

hash_t** HashTableCreate(void);
int HashTableInsertData(hash_t** t, datatype data);
void HashTableShow(hash_t** t);
int HashTableSearchData(hash_t** t, datatype data);

hash_t** HashTableCreate(void)
{
    hash_t** t;

    t = (hash_t**)malloc(N * sizeof(hash_t*));
    if (t == NULL) {
        printf("%s malloc memory error\n", __func__);
        return NULL;
    }

    for (int i = 0; i < N; i++) {
        t[i] = (hash_t*)malloc(sizeof(hash_t));
        if (t[i] == NULL) {
            printf("%s malloc memory error\n", __func__);
            return NULL;
        }
        t[i]->data = (datatype)0;
        t[i]->next = NULL;
    }

    return t;
}
int HashMembInsert(hash_t* h, datatype data)
{
    hash_t* tmp;

    tmp = (hash_t*)malloc(sizeof(hash_t));
    if (tmp == NULL) {
        printf("%s malloc memory error\n", __func__);
        return -1;
    }
    tmp->data = data;

    tmp->next = h->next;
    h->next = tmp;
    return 0;
}
int HashTableInsertData(hash_t** t, datatype data)
{
    return HashMembInsert(t[data % N], data);
}
void HashMembShow(hash_t* h)
{
    while (h->next) {
        printf("-%d", h->next->data);
        h = h->next;
    }
}
void HashTableShow(hash_t** t)
{
    for (int i = 0; i < N; i++) {
        printf("t[%d]", i);
        HashMembShow(t[i]);
        printf("-\n");
    }
}
int HashMembSearchData(hash_t* h, datatype data)
{
    int pos = 0;
    while (h->next) {
        if (h->next->data == data)
            return pos;
        pos++;
        h = h->next;
    }

    return -1;
}
int HashTableSearchData(hash_t** t, datatype data)
{
    return HashMembSearchData(t[data % N], data);
}

#define ARRAY_SIZE(arr) (sizeof(arr) / sizeof(arr[0]))

int main(int argc, const char* argv[])
{
    hash_t** t;
    int k[] = { 23, 34, 14, 38, 46, 16, 68, 15, 07, 31, 26 };

    t = HashTableCreate();
    if (t == NULL)
        return -1;

    for (int i = 0; i < ARRAY_SIZE(k); i++) {
        HashTableInsertData(t, k[i]);
    }

    HashTableShow(t);

    int num = 46;
    printf("HashTable search t[%d]:data=%d : pos=%d\n",num%N,num,HashTableSearchData(t,num));
    return 0;
}
